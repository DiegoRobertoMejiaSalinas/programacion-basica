COMO INSTALAR WOOCOMMEERCE EN WORDPRESS
https://www.youtube.com/watch?v=LeFnrLfdiuA


EN ESTE DEMO TENEMOS QUE USAR FUNCIONALIDADES DE LA WEB.
OPCION 1:
EN ESTE CASO WOOCOMMERCE ES UN PLUGIN DE WORDPRESS PARA CREAR TU TIENDA ONLINE, LO QUE HAREMOS SERA LLAMAR A LOS PRODUCTOS DE LA TIENDA Y MOSTRARLO EN NUESTRA APLICACION ANDROID(SE VE MAS ORIENTADO A LO QUE SERIA UN MODELO DE NEGOCIO)
AQUI TENDRIAMOS QUE LLAMAR AL API REST DE WOOCOMMERCE

OPCION 2:
EXTRAER LAS PUBLICACIONES QUE TENGAMOS EN NUESTRA PAGINA Y MOSTRARLA EN NUESTRA APLICACION ANDROID
PARA ESO USAREMOS EL API REST DE WORDPRESS



POSDATA: 
APARTIR DE LA VERSION 5.1 DE WORDPRESS YA VIENE PREINSTALADO EL API REST DE WORDPRESS

APARTIR DE LA VERSION 2 DE WOOCOMMERCE YA VIENE EN SUS CONFIGURACIONES INSTALADO EL API REST DE WOOCOMMERCE
